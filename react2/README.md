# React 12.2 シンプルアプリケーション

特別なサーバーの用意が不要な、シンプルなReact 12.2プロジェクトです。

## 特徴

- React 12.2を使用
- 特別なサーバーが不要
- index.js等の内容が少ない
- シンプルな構成

## セットアップ

1. 依存関係のインストール:
```bash
npm install
```

2. 開発サーバーの起動:
```bash
npm start
```

3. ブラウザで http://localhost:3000 を開く

## ビルド

本番用ビルド:
```bash
npm run build
```
